# Input data (BIDS)

Place T1w MRI here in [BIDS format](https://bids.neuroimaging.io/).

```text
input/
  bids_config.json
  dataset_description.json
  sub-<subject_id>/anat/sub-<subject_id>_T1w.nii.gz
```

Copy the `.example` JSON files and see [USER_GUIDE.md](../USER_GUIDE.md#preparing-input) for session configuration and demographics.
